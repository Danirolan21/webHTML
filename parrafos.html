Párrafos

Los parrafos se hacen con la etiqueta <p>

Los parrafos tienen su propio diseño. Entre dos párrafos siempre habrá una linea en blanco

Si alguna vez quereis bajar de linea utilizamos la etiqueta <br>

Aqui teneis un ejemplo. Voy a bajarme de linea.
Ya estoy en la linea de abajo.

Cuando simplemente queremos ponerle un estilo a un par de palabras utilizamos la etiqueta <span>
Aqui se puede ver un ejemplo.

En contrapartida existe otra etiqueta de tipo <div> de tipo bloque que sirve para encerrar cualquier contenido.

Para hacer una negrita utilizaremos la etiqueta <strong>. Aqui teneis una negrita.

Para hacer una cursiva utilizaremos la etiqueta <em>. Aqui teneis una cursiva.

Para hacer un subindice utilizaremos la etiqueta <sub>. Aqui teneis un subindice x2

Para hacer un superindice utilizaremos la etiqueta <sup>. Aqui teneis un superindice x2

Para hacer una abreviatura utilizaremos la etiqueta <abbr>. Aqui teneis una abreviatura HTML
